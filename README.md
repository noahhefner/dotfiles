# Linux Desktop

Configuration files, applications, and setup for my Linux desktop. Tailored for my Lenovo Thinkpad T430.

![desktop screenshot](screenshot.png)

## Installation

Execute scripts:

```sh
./setup.sh
```

```sh
./install.sh
```

```sh
./configs.sh
```

```sh
./post-setup.sh
```

Reboot the machine, then run:

```sh
./post-reboot.sh
```
